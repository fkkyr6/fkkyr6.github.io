英语词根词缀

英语 词根词缀
英语 词根词缀
发布日期:   2021-04-16
更新日期:   2021-04-16
文章字数:   3.1k
阅读时长:   11 分

考研常用词根词缀表

a开头

a- 加在单词或词根前面, 表示不，无，非
a- 加在单词前, 表示在..., ...的
ab-, abs- 加在词根前,表示相反，变坏，离去等
ab-, ac-, ad-, af-, ag-, an-, ap-, ar-, as-, at-等 加在同辅音字母词根前，表示一再等加强意
ad- 加在单词或词根前, 表示做....., 加强...... 
amphi- 表示两个、两种
an- 在词根前, 表示不，无
ana- 表示错误，在旁边，分开
ante- 表示前面，先
anti- 表示反对，相反
apo- 表示离开，远离
auto- 表示自动、自已
-ability  表名词， 能...；性质 
-able  表形容词， 可...的，能... 
-ably  表副词， 能...地   
-aceous  表形容词， 具有...特征的 
-acious  表形容词， 有特征的，多...的 
-acity  表名词， 有...倾向   
-acle  表名词,   ... 物品，状态  
-acy  表名词,   ...性质，状态 
-ad  表名词， ...东西，状态
-ade  表名词, 表示 状态，物品  
-ade  表示 个人或集体 
-age  表示： 费用 
-age  表示 场所，物品   
-age  名词后缀,表示 状态，总称  
-ain  表名词， ...人 
-air  表名词， 人、物  
-aire  表名词， ...人 
-al  表形容词， ...的 
-al  表名词， 人，物，状态 
-ality  表名词， 状态，性质  
-ally  表副词，由al+ly 构成， ...地  
-an  表名词和形容词， ...地方，...人 
-ance  表名词， 性质，状况  
-aneity  表名词，表示 性质，状态  
-aneous  表形容词， ...有; ...特征的  
-ant  表形容词， ...的 
-ant  表名词， ...人  
-ant  表名词， ...剂 
-ar  表形容词， ...的 
-ard  表名词， 不好的人   
-arian,  表形容词或名词， ...的(人) 
-arium  表名词， 地点，场所  
-ary  表名词， 人，场所，物  
-ary  表形容词， ...的 
-ast  表名词， ...人，物   
-aster  表名词， 不怎么样的人 
-ate  表动词， 做，造成
-ate  表形容词， 具有...的 
-ate  表名词， 人或地位
-atic  表形容词， 有...性质的 
-ation  表名词， 行为，过程，结果
-ative  表形容词， 有...倾向(性质)的   
-ator  表名词，通常由ate结尾的动词而来， 做事的人或物 
-atory  表名词 场所，地点
-atory  表形容词， 有...性质的

b开头

be- 构成动词，表示使......成为
be- 构成一些介词
bene- 表示善, 好
bi- 表示二个, 两
by- 表示在旁边，副的

c开头

cata- 表示向下，相反，离开
circum- 表示环绕，周围
co- 表示共同，通常放元音词根前
col-, cor- 在同辅音词根前, 表示   共同
com-, con- 表示共同
contra- 表示反对，相反
counter- 表示反对，相反   
-cy  表名词，也作-acy,   性质，状态

d开头

de- 表示去掉，变坏，离开，变慢，向下等
de- 表示使....成为，加强等
deca- 表示十
deci- 表示十分之一
demi- 表示半
di- 表示二个，双   
di- 表示使...变成，分开，离开
dia- 表示穿过，二者之间   
dif- 和辅音重复表示不，否定，分开
dis- 表示不，消失掉
dis- 表示分开，分离
dys- 表示坏，不良
-dom  表名词， 状态或领域

e开头

e-, ef- 表示出，出来，   
em-, en- 表示进入... 之中，包围
em-,en- 表示使..... 进入状态
endo- 表示内部   
epi- 表示在...上，在...周围，在...后面
eu- 表示好，优秀
ex-  表示   出，出去
ex-  表示   前面的，前任的
exo- 表示   外部的，外面
extra-  表示   以外的，超过的
-ee  表名词， 被动或主动的人
-eer  表名词， ... 人员
-el  表名词， 人或物 
-en  表动词， 变成   
-en  表形容词， 由...制成的 ，通常加在名词后面
-en  表名词， 人或物 
-ence  表名词， 性质，状态
-ency 表名词，ence 的变体
-enne  表名词， 女性
-ent  表形容词， ...的   
-ent  表名词， ...药剂
-ent  表名词， ...人 
-eous  表形容词， 有...的 
-er  表动词， 反复做
-er  表名词， 物品，机器
-er  表名词， ...人
-ern  表形容词， ...方向的
-ern  表名词， ...场所
-ery  表名词， 场所，地点
-ery  表名词， 行为，情况
-esque  表形容词， 如...的
-ess  表名词， 女性，雌性
-et  表名词， 小东西
-etic  表形容词， 属于...的
-ette  表名词， 小的东西或状态
-ety  表名词， 状态
-eur  表名词， ...人

f开头

fore-  表示   前面，预先
-faction  表名词， 达到的状态 ，由-fy转化而来
-fic  表形容词， 产生...的
-fication 表名词，由fic变化而来
-fier  表名词， 人或物 ，由-ify转化而来
-fold  表形容词或副词， 倍，双重   
-form  表形容词， 有...形状的
-ful  表形容词, 有...的
-ful  表名词, 满，量   
-fy  表动词, ...化，成为... ，更多时候作 -ify

h开头

hecto-  表示   百，许多
hemi-  表示   半 
hepta-  表示   七   
hetero-  表示   异类，异种
hexa-  表示   六
holo-  表示   全部
homo-  表示   同类的
hyper-  表示   超过，太多
hypo-  表示   下面，次等
-hood  表名词, 时期，性质等

i开头

il-, ir-  辅音重复表示   不，无
il-,ir-  表示   使.....成为，进入
im-, in-  表示   不，无，非
im-,in-  表示   向内，进入
inter-  表示   在..... 之间，相互
intra-  表示   在内，内部
intro-  表示   向内，入内
iso-  表示   等, 同 
-ia  表名词, 某种病 
-ia  表名词, 总称，状态 
-ial  表形容词， 有...的 
-ian  表名词， 某种人
-ian  表形容词,   ...国家的
-ibility  表名词， 具备...性质的
-ible  形容词   能...的   
-ic  表形容词,   ....的 ，有时作-tic
-ic  表名词， 某种药
-ic  表名词， 人或学科
-ical  表形容词， ...的 
-ice  表名词， 行为，状态
-ics  表名词， 学科，学术
-id  表形容词， 如...的 
-ie  表名词， 小东西或人
-ier  表名词， 人或物 
-ile  表形容词， ...的   
-ile  表名词， 物体
-ine  表形容词， ...的
-ine  表名词， 人或女人
-ine  表名词， 状态, 药物等
-ing  表形容词， 正...的 ， 令人...的
-ing  表名词， 行业
-ing  表名词， 物品
-ing  表名词， 状态   
-ion  表名词， 动作或状态 分为 --sion和--tion两种
-ion  表示 某种物，用品
-ior  表形容词， 较...的
-ious  表形容词, ....的   
-ise  表动词后缀，和 -ize 相同，是 -ize 的变体， ...化 
-ise  表名词， 物品，状态   
-ish  表形容词， 象...一样，有的... ，通常放在一具体名词后
-ish  表动词， 造成... 
-ish 表示国家的或语言
-ism  表示   学术或学术流派
-ism  表示   行为, 现象，状态
-ism  表示   疾病 
-ism  表示   具备某种性质
-ism  名词，表示   各种主义，宗教   
-ist  表名词， 表示信仰者，专家或从事人   
-ist 以-ist 结尾的单词，不少同时可做名词和形容词用。
-istic  表形容词，由-ist + ic 构成,表示    ...的
-it 表名词，通常是抽象名词
-it  表名词， ...人 
-ite  表形容词， 有...的
-ite  表名词， 人或物
-ite  表动词， 促成   
-ition  表名词， 行为过程，状态   
-itious  表形容词，-ition变化而来，表示   ...的 
-itive  表形容词， ...的  
-itor  表名词， 人   
-itude  表名词， 性质，状态等 
-ity 表名词，指具备某种性质
-ive  表形容词，根据不同的单词结尾可分-sive, -ative, ive, itive
-ivity  表名词，由-ive+ity 组成， 有...能力或特性 
-ization  表名词，来自-ize+ation,   ...化 或  发展过程 
-ize 构成一些动词

k开头

kilo-  表示   一千
-kin  表名词， 小...   
-less  表形容词，  无...的，不...的
-let = -et  表示 小
-like  表形容词， 像...一样   
-ling  表名词，  小东西或某种人
-ly 表形容词，通常加在名词后
-ly 表副词，通常放在形容词后

l开头

-less  表形容词，  无...的，不...的
-let = -et  表示 小
-like  表形容词， 像...一样   
-ling  表名词，  小东西或某种人
-ly 表形容词，通常加在名词后
-ly 表副词，通常放在形容词后

m开头

macro-  表示   宏传， 大
mal-; male  表示   坏，恶
meta-  表示   超过, 改变
micro-  表示   微，小
milli-  表示   千，千分之一
mini-  表示   小 
mis-  表示   错误，坏 
mono-  表示   单个，一个 
multi-  表示   多，很多 
-ment  表名词， 行为或结果
-ment 表示具体物  
-most  表形容词， 最...的

n开头

neo-  表示   新的
non-  表示   不，非   
-ness  表名词， 性质, 状态， 通常加在形容词后面

o开头

ob-  表示   逆，倒，加强意义   
octa-  表示   八 ; 亦作octo 
omni-  表示   全部，到处 
out-  表示   超过，过度  
out-  表示   出去，过时  
over-  表示   过度，过份 
over-  表示   翻转  
over-  表示   在... 之上  
-o  表名词， 人，物或状态
-o  表名词，用于音乐术语  
-on 表名词，指人、物和一些物理学上的名词 
-oon  表名词， 人或物   
-or  表名词， 人或物器 ，在一些词根前写成 -ator
-orium = arium  表名词， 地点，场所
-ory 表名词，指场所等  298  -ory  表形容词， ...的 
-ose  表形容词， 多...的   
-osity  表名词，-ose+ity组成，表示 多...的状态
-ot  表名词， ...人   
-ous  表形容词，通常放在一个完整的单词后，表示 ...的

p开头

paleo-  表示   古，旧  
pan-  表示   广泛的   
para-  表示   半，类似，辅助  
para-  表示   旁边  
para-  表示   降落伞 
pen-  表示   近似，差不多 
penta-  表示 五   
per-  表示   贯穿，自始至终 
per-  表示   假，坏  
peri-  表示   周围，靠近
poly-  表示   多 
post-  表示   在后面 
post-  表示   邮件，邮政 
pre-  表示   ...前的，预先 
pro-  表示 赞同，亲...  
pro-  表示   向前，在前  
pro-  表示   很多... 
proto-  表示   原始... 
pseudo-  表示   假，伪  
-proof  表形容词， 防...的

q开头

quadri-,quadru-  表示   四 
quasi-  表示   类似，准

r开头

re-  表示   向后，相反  
re-  表示   一再，重新  
retro-  表示   向后，倒退   
-ress  表名词，指 物品
-ress  表名词，指 女性
-ry 表示集合名词 
-ry  表示   场地 
-ry  表示 行业，学科   
-ry 表名词，通常放在一个完整的单词后面表示状态，性质

s开头

se-  表示   分开，离开，区别开
semi-  表示   半 
sept-,septi-  表示   七  
sex-, sexi-  表示   六   
step-  表示   后，继或前夫(妻)所生
stereo-  表示   立体   
sub-  表示   在下面，次一等，副手 
sub-  表示   接近，靠近   
suc-, suf-, sup-, sur-  等辅音重复表示    在...下面 
super-  表示   在...上面   
super-  表示   超级，超过，过度
supra-  表示   超...   
sur-  辅音不重复表示   超过，在上面
sus-  表示   在... 下面 
sym-, syn-  表示   共同，相同
-ship 表示某种技能  
-ship  表名词,表示某种关系或状态  
-some  表形容词， 充满...的，具有...倾向的 
-ster  表名词， ...人

t开头

tetra-  表示 四   
trans-  表示   横过，越过
trans-  表示   变换，改变；转移
tri-  表示   三 
twi-  表示 二、两  
-th 表名词，通常指抽象名词  
-tic  表形容词，通常放在一个名词前， 与...相关的，...的 
-ture -ture在s结尾的词根后变成-sure  
-ture  表名词，通常在单词或词根以t结尾时使用，表示 一般状态, 行为 并作 -ature  iture   
-ty 表名词，用在形容词后，把形容词变成名词

u开头

ultra-  表示   极端 
ultra-  表示   超出，超过   
un-  表示   不，无，非，没有 
un-  表示   打开，解开，弄出
under-  表示   在...下，在...之内 
under-  表示   不足，不够  
under-  表示   副手
uni-  表示   一个、单一 
-ular  表形容词, 有...形状或性质的 
-ule  表名词， 小... 
-uous  表形容词， 多...的

v开头

vice-  表示   副

w开头

with-  表示   向后，相反  
-ward  表形容词+副词， 向...   
-wise 有些单词，wise可以换成ways，意义不变 
-wise  表副词， 方向，状态

y开头

-y 表名词，加在形容词或以r结尾的单词后。
-y 表名词， 人或小东西 常带有嬉谑性和爱称。
-y 表形容词，加在名词后变成形容词

补充

ancy = ance  表示  性质，状况  
aer = air空气(元音变化) aerial空气的 / aeroplane飞机 
alt = high高(Altai阿尔泰山) altitude高度 
annu = year年(an一n年) anniversary周年纪念 / annual年度的
arch =长者、统治 anarchy无政府状态 / monarchy君主政体
astro = star星 astrology占星术 / astronomy天文学
audi =hear听 audience听众 / audit旁听、审计
auto-=self自我 autocrat专制者 / automobile汽车 
bio = life生命(bio=body) biology生物学 / biotic生命的、生物的
cardi = heart心 cardiac心脏的 / cardinal衷心的
cess = go行走 precede领先/exceed超过 / process过程
cert = certain确定的 certify证实 / certificate证明书
circ=ring环形 circus马戏场 /circulate流通、流传
cosm = world 世界 cosmic宇宙的 
cred = believe相信 credible可信的/credulous轻信的
dent = tooth牙齿 dentist牙医 / trident三叉戟
dict = say说 dictation口述、听写 / dictator独裁者

整理于网络，如发现有误，请留言。
